## Ceifadora - Convocatórias

Este projeto é uma aplicação frontend construída com Vite, React, TypeScript, Tailwind CSS e shadcn-ui.

### Requisitos

- Node.js e npm instalados (recomendado usar nvm)  
  Veja: [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Como rodar o projeto localmente

```sh
# Instalar dependências
npm install

# Rodar em modo desenvolvimento
npm run dev
```

### Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Build para produção

```sh
npm run build
```
